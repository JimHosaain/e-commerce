
1. null vs undefined: null mane variable intentionally khali kora, ar undefined mane variable define kora hoyeche kintu kono value deya hoyni..

2. map(): map() array-er prottek element e function apply kore notun array return kore.. kintu forEach() sudhu loop chalay.. kono return dey na..

3. == vs ===: == shudhu value check kore ar type ignore kore.. kintu === value o type duita check kore.

5. Scope: JavaScript e variable kothay access kora jabe ta define kore; global (shob jaigai), function (function-er vitore), block (if/for block-er vitore) scope thake.
